How to use TCG Card Tracker

Adding cards

Click + Add card via search at the bottom of the table. Type a card name — the search tries an exact match first, then falls back to a wildcard if nothing is found. Optionally enter a set name to narrow results. Toggle Japanese cards to search via TCGdex (real Japanese set data, TCGPlayer USD prices). Toggle Promo only to filter English results to promo cards. Click a result to select a finish, then press Add to list or Quick add — the modal stays open so you can keep adding. You can also paste a TCGPlayer product URL in the second tab.

Editing cards

Click the card image to open the quick-edit popover — a compact panel where you can change condition, finish, buy cost, market price, and notes without scrolling the table. You can also edit Buy cost and Sold price inline; press Enter or click away to save. The Duplicate button (⧉) in the actions column inserts a copy below, useful for multiple copies of the same card in different conditions.

Tracking sales

Check the Sold checkbox on the right of a row — the row turns green. Enter the actual sale price in Sold price and the Profit column switches from Expected (market-based) to Actual (sold price − buy cost). Total sold and Actual profit in the summary bar update in real time.

Refreshing prices

Click Refresh prices to pull the latest market prices from pokemontcg.io. A progress bar shows the current card being fetched and how many remain. Cards with the same API ID share one fetch (deduplication). Check Skip sold to skip sold cards and save API calls. After refresh, the Δ Price column shows ▲/▼ per card, and the Expected profit summary shows the net portfolio delta since the last refresh. Prices are cached for 1 hour — click Clear cache to force a full re-fetch. New sets may show no price data for a few days until TCGPlayer's feed updates.

Bulk actions

Check one or more row checkboxes to reveal the bulk toolbar. Mark as sold marks all selected cards at once. Delete selected removes them after confirmation. The header checkbox selects all currently visible (filtered) rows. All bulk actions are undoable.

Filtering, sorting & columns

Type in the filter bar to instantly search your list by card name or set name. Click any column header to sort — click again to reverse. Use Columns ▾ to hide columns you don't need; preferences persist across sessions. The Image, Name, Sold, and action columns are always visible.

Import & export

Export CSV downloads your full list. Import CSV loads it back — choose to add to the existing list or replace it. Only a name column is required; everything else defaults gracefully, so a simple spreadsheet with name and buy cost is enough to get started.
